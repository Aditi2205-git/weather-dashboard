# 🌦 Weather Dashboard

A JavaScript weather dashboard that fetches weather for multiple cities using Promise.all and the Open-Meteo API.

## Features

- Parallel API requests with Promise.all
- Weather cards UI
- Glassmorphism design
- Loading spinner
- Error handling

## Technologies

- HTML
- CSS
- JavaScript
- Open-Meteo API

## Cities

- Delhi
- London
- Tokyo
